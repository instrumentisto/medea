Medea Control API protocol
==========================

Control API protocol implementation for [Medea] media server.

__Currently, in early development phase.__




## Prerequisites

- CMake >= 3.8.0
- Rust >= 1.19.0
- binutils >= 2.22
- LLVM and Clang >= 3.9
- protoc >= 3.7 (for rebuilding [protobuf] specs)
